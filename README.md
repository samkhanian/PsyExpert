# MindRules — Simple Expert System for Personality Patterns

[فارسی](./README.fa.md) · [Demo](https://samkhanian.github.io/PsyExpert/)

**Student:** Jamal Samkhaniyan  
**Instructor:** Dr. Roya Namiranian  
**Course:** Expert Systems

MindRules is a lightweight, rule-based Expert System implemented with HTML, CSS, and JS.  

## Purpose

- Knowledge base with rules  
- Forward-chaining inference engine  
- Interactive user interface  
- Explainability for reasoning

## Architecture

- `index.html` — main page with UI  
- `style.css` — styling  
- `rules.js` — questions and rules  
- `engine.js` — inference engine  
- `README.md` — English documentation  
- `README.fa.md` — Persian documentation

## Run

Open `index.html` in a browser, answer the questions, click **Run**, and see the results along with explanations and JSON output.
