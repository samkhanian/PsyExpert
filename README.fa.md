# MindRules — سیستم خبره برای الگوهای شخصیتی

**دانشجو:** جمال سام خانیان  
**استاد:** دکتر رویا نمیرانیان  
**درس:** سیستم‌های خبره

MindRules یک سیستم خبره سبک با قواعد آماده است که با HTML, CSS و JS پیاده‌سازی شده است.  

## هدف

- پایگاه دانش با قواعد  
- موتور استنتاج زنجیره پیشرو  
- رابط کاربری تعاملی  
- نمایش علت تصمیم‌گیری (Explainability)

## معماری

- `index.html` — صفحه اصلی و رابط کاربری  
- `style.css` — استایل  
- `rules.js` — سوالات و قواعد  
- `engine.js` — موتور استنتاج  
- `README.md` — مستندات انگلیسی  
- `README.fa.md` — مستندات فارسی

## اجرا

فایل `index.html` را در مرورگر باز کنید، سوالات را پاسخ دهید، روی **بررسی و اجرا** کلیک کنید و نتایج و توضیحات JSON را ببینید.
