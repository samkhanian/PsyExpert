# MindRules --- سیستم خبره آکادمیک برای بررسی گرایش‌های شخصیتی

[English](./README.md) · [فارسی](#)

**دانشجو:** جمال سام خانیان  
**استاد:** دکتر رویا نمیرانیان  
**درس:** سیستم‌های خبره

MindRules یک سیستم خبرهٔ ساده و آموزشی بر پایهٔ قواعد است. این پروژه برای اهداف دانشگاهی طراحی شده و ابزار تشخیص بالینی نیست.

## هدف

نمایش:  
- پایگاه دانش مبتنی بر قواعد  
- موتور استنتاج زنجیره پیشرو  
- رابط کاربری ساده  

## معماری

### پایگاه دانش

نمونه قواعد:

اگر فرد تنهایی را ترجیح دهد و جداشدگی عاطفی مداوم داشته باشد  
آنگاه الگوی «شیزوئید گونه» محتمل است.

اگر روابط ناپایدار و رفتارهای تکانشی داشته باشد  
آنگاه الگوی «مرزی‌گونه» محتمل است.

### موتور استنتاج

مراحل:  
1. پرسیدن سؤال  
2. دریافت پاسخ  
3. تطبیق قواعد  
4. امتیازدهی  
5. خروجی + قواعد فعال‌شده  

### رابط کاربری

صفحهٔ HTML با:  
- سؤالات  
- نتیجه  
- خروجی JSON
